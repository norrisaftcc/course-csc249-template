What You're Seeing

Each creature in the petri dish is an object — an instance of the Critter class. It has state (properties like value, pos, vel) and behavior (methods like wander(), separate(), display()).

Notice how each critter's shape (number of sides) and size map to its value. Low values are small triangles; high values are large octagons. The object bundles all this data together — just like the Course class from Module 1.

Try: Click a critter to select it. Change the Speed slider to 0 and watch it stop. Then ask yourself: what does maxSpeed control in update()?
